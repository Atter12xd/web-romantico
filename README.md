### See it in action

[astro-frieren-blog-kit.netlify.app](https://astro-frieren-blog-kit.netlify.app)

## Deploy to Production

[Deploy to Netlify]

### Or clone this repo

```
git clone https://github.com/fishingelephants/astro-frieren-blog-kit frieren-blog
```

```
cd frieren-blog
```

```
npm install
```

```
npm run dev
```
